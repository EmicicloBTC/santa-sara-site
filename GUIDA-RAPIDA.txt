SANTA SARA — Guida rapida (senza terminale)

1) Devi avere Node.js installato (gratis): https://nodejs.org — scegli "LTS", installa, riavvia il PC se serve.

2) Anteprima sul tuo computer:
   - Doppio clic su: AVVIA-SITO.bat
   - Nel browser apri l'indirizzo che compare (es. http://localhost:5173)

3) Concept attuale del sito:
   - Una sola schermata fullscreen con foto editoriale (catalogo).
   - Sopra ai prodotti in vendita ci sono "hotspot" cliccabili (puntino con +).
   - Click sul puntino = si apre il modal con foto, dimensioni, prezzo, CTA.
   - Frecce sinistra/destra (o swipe su mobile) per cambiare scena.
   - Header in alto con logo + menu (Bio, Instagram, Contatti).
   - Footer in basso con titolo scena + numero scena (es. 01 / 02).

4) Cambiare le foto delle SCENE:
   - Mettile in: public\images\scenes\
   - Per ogni scena servono DUE foto: una desktop (orizzontale) e una mobile
     (verticale). Esempio:
       desktop:  scene-1.png         (1920x1080 o simile)
       mobile:   scene-1-mobile.png  (1080x1920 o simile)
     Se non hai ancora la mobile, lascia stare: il sito usa la desktop come
     fallback (su telefono però l'immagine viene croppata ai lati).
   - Apri src\data\scenes.js e per ogni scena imposta:
       image:        "/images/scenes/scene-1.png"
       imageMobile:  "/images/scenes/scene-1-mobile.png"   (opzionale)
       title:        "..."        (compare nel footer)
       hotspots:        [ { productId: "florence", x: 38, y: 52, r: 6 }, ... ]   // versione desktop
       hotspotsMobile:  [ { productId: "florence", x: 38, y: 60, r: 7 }, ... ]   // versione mobile (opzionale)

5) Posizionare i HOTSPOT (i puntini cliccabili sopra ai prodotti):
   - Avvia il sito (AVVIA-SITO.bat).
   - Sulla landing premi "E" (oppure "?") sulla tastiera: si apre il pannello
     editor in alto a destra. In alto al pannello trovi un'etichetta:
        "Desktop · orizzontale"  oppure  "Mobile · verticale"
     che ti dice in che modalità sei. Per testare la mobile rimpicciolisci la
     finestra del browser fino a farla diventare verticale (oppure ruota il
     telefono).
   - Nel pannello scegli il bottone con il nome del prodotto (Florence/Romance).
   - Clicca sulla foto sopra al prodotto: compare una riga di codice già pronta.
   - Premi "Copia riga" e incollala in scenes.js dentro:
        hotspots: [...]        se eri su desktop
        hotspotsMobile: [...]  se eri su mobile
     (te lo ricorda anche il pannello: "3. Incolla in scenes.js dentro: …").
   - Premi di nuovo "E" per chiudere l'editor.

6) Cambiare i PRODOTTI:
   - Foto: cartella public\images\products\NOME-PRODOTTO\1.jpg, 2.jpg, ...
     (NOME minuscolo, senza spazi)
   - Dati: src\data\products.js — aggiorna title, kind, dimensions, description,
     price, cta, ctaHref e images: { slug: "nome", count: N, ext: "jpg" }.
   - Per usare un prodotto in una scena, metti la sua chiave (es. "florence")
     in productId dentro hotspots.

7) Mettere online le modifiche:
   - Se hai collegato GitHub a Netlify (modalità attuale): apri Cursor o terminale,
       cd c:\Users\pucci\Desktop\CODICI\santa-sara-site
       git add .
       git commit -m "messaggio"
       git push
     Netlify rifa il deploy da solo in 1-2 minuti.
   - Solo trascina cartella: doppio clic CREA-PER-NETLIFY.bat e trascina la
     cartella "dist" su Netlify.

8) Se qualcosa non funziona, copia l'errore della finestra nera in chat.
